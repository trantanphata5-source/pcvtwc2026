# Antigraviti PCVT - CHÁY HẾT MÌNH CÙNG WORLD CUP 2026

Web app cá cược vui vẻ nội bộ World Cup 2026 - Công ty Điện lực Vũng Tàu.

## Tính năng
- 📅 Lịch thi đấu 104 trận WC 2026 (48 đội chính thức)
- 🏟️ Bảng đấu + Nhánh knockout
- 🎯 Đặt cược (20.000đ/trận, handicap)
- 💰 Ví tiền + Nạp tiền
- 🏆 Bảng xếp hạng
- 🛡️ Admin panel

## Tech Stack
- **Frontend**: Vue 3 + Tailwind CSS + Lucide Icons
- **Backend**: Google Apps Script
- **Database**: Google Sheets
- **Deploy**: Vercel (frontend) + Google Apps Script (API)

## Deploy lên Vercel
1. Push code lên GitHub
2. Import project trên [vercel.com](https://vercel.com)
3. Deploy (chỉ cần file `Index.html` và `vercel.json`)
